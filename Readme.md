# Programming Advices

This repository contains the path I followed to learn programming and its foundations in a healthy way, presented by **Dr. Muhammad Abu Hudhud**, who has more than **29 years of experience** in the field of programming and management, and is the founder of the **Programming Advices** platform.
platform link  https://programmingadvices.com

> **Note:** This repository only contains my practical solutions and projects.  
> It does not include the theoretical courses that do not involve writing code.

---

## Programming Advices Course Details

### Course 01 – Important Basics for Every Programmer (Level 1)
- The computer and its components  
- CPU and GPU  
- Methods of representing data in the computer  
- Methods of representing letters and units of measurement in the computer  
- Methods of converting between numerical systems  
- Networks  
- Types of programming languages, their speeds and uses  
- Difference between compiler and interpreter  
- Types of operations in the computer and calculation priorities  

---

### Course 02 – Algorithms and Problem Solving (Level 1)
- Contains **50 simple problems** solved using a flowchart.

---

### Course 03 – C++ (Level 1)
- Covers the basics of C++ programming, such as:
  - Variables  
  - Functions  
  - Loops  
- Each lesson has an assignment to apply what you have learned by solving problems from Course 02.

---

### Course 04 –  Algorithms and Problem Solving (Level 2)
- Solving Course 02 problems in **more advanced ways**.  
- Learn how to write clean code.  
- Apply the **Divide and Conquer** principle.

---

### Course 05 – Algorithms and Problem Solving (Level 3) + Three Projects
- Another set of **50 problems** of varying difficulty.  
- Includes two mini-projects:
  1. Rock, Paper, Scissors  
  2. Quiz Game  
- **Additional Project (added by me):**  
  A calculator for converting between number systems (binary, hexadecimal, etc.).

### Course 07 – Algorithms and Problem Solving (Level 3)
- Contains **51 problems** of varying difficulty.  
- Topics include:
  - Matrix  
  - Text handling  
  - File handling  
  - Vectors

  1. **Bank Project (Part 1):** Adding, deleting, modifying, and viewing customers.

### Course 08 – Algorithms and Problem Solving (Level 4)
- Contains **65 problems** of varying difficulty.

  - **String manipulation**
